伊東の見どころ＆名物

名物料理：うずわめし / 干物 / 金目鯛
  • うずわめしはソウダガツオを薬味とたたいた漁師飯。醤油→ご飯→だし茶漬けで味変するのが定番。
  • 干物は伊東の定番。凍ったまま焼くと旨味が逃げにくい。
  • 金目鯛は煮付け・刺身・寿司が王道。誕生日感も出る。

出典: Tabirai
